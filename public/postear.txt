POSTE.AR - MANUAL DE USO
========================

POSTEAR
-------
Escribe tu mensaje (máximo 576 caracteres) y presiona [postear].
Tu marca temporal aparecerá en todos tus posts durante esta sesión.

FORMATO DE TEXTO
----------------
*texto*              → texto en negrita
_texto_              → texto en cursiva
<nombre:url.com>     → crear enlace

Ejemplo:
Hola *mundo*, esto es _importante_. 
Más info en <sitio:ejemplo.com>

OCULTAR CONTENIDO
-----------------
((secreto|clave123))

El texto entre (( )) se cifra con la semilla "clave123".
Solo quien conozca la semilla podrá descifrar el contenido.

NOTA: Función de cifrado próximamente disponible.

REETIQUETAR
-----------
Presiona [reetiquetar] para obtener una nueva marca anónima.
Útil si quieres cambiar de identidad sin cerrar sesión.

TU MARCA
--------
Formato: xxxx.NN.yyyy.MM
Ejemplo: tide.21.yija.87

Es única y temporal. No se guarda en ningún servidor.

PERSISTENCIA
------------
Los mensajes se eliminan automáticamente cada 7 días
para mantener la base de datos efímera y renovada.

PRIVACIDAD
----------
- No se registran IPs ni datos personales
- No hay cuentas de usuario
- Todo es anónimo y temporal
- El código es open source
