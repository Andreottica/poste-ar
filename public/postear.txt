POSTE.AR - MANUAL DE USO
========================

Red social anónima con identidad persistente.
Sin registro, sin IPs, sin datos guardados.
Todo se borra cada lunes a las 00:00 hs.


TU IDENTIDAD ANÓNIMA
====================

CÓMO CREAR TU USUARIO
---------------------
Al entrar por primera vez:

1. Escribí 3-5 palabras (ej: perro mesa luz golf)
2. Presioná [ entrar ]
3. El sistema genera tu usuario anónimo (ej: meno.34.rite.87)

IMPORTANTE: Tus palabras SON tu identidad.
- Si las recordás → siempre recuperás el mismo usuario
- Si las olvidás → perdés tu identidad para siempre
- Si las compartís → otra persona puede ser vos

NO GUARDAMOS NADA
-----------------
- No hay registro
- No pedimos email ni teléfono
- No guardamos tus palabras
- No rastreamos IP
- Solo VOS conocés tu identidad

RECUPERAR TU USUARIO
--------------------
Para volver a ser el mismo usuario en otra sesión:

1. Ingresá las MISMAS palabras que usaste antes
2. El sistema genera el MISMO usuario
3. Podés postear con tu identidad habitual

CAMBIAR DE IDENTIDAD
---------------------
Presioná [ salir ] y entrá con palabras nuevas.
Cada combinación de palabras = usuario único.

RECOMENDACIONES
---------------
- Usá 3-5 palabras fáciles de recordar
- Ejemplo: "gato verde tiza luna"
- Anotá tus palabras en papel (NO en computadora)
- Nunca uses menos de 3 palabras (muy inseguro)


CÓMO PUBLICAR
=============

MENSAJE PÚBLICO
---------------
1. Escribí tu mensaje (máximo 576 caracteres)
2. Presioná [ postear ]
3. Aparece en el timeline con tu usuario

MENSAJE OCULTO CIFRADO
-----------------------
Para agregar un mensaje secreto:

1. Activá "Agregar mensaje oculto cifrado"
2. Escribí tu mensaje secreto (máximo 288 caracteres)
3. Escribí una contraseña (ej: "rosa viento mar")
4. Presioná [ postear ]

El mensaje público se ve normalmente.
El mensaje oculto queda cifrado (ilegible).

MENSAJES INVISIBLES
--------------------
Podés postear SOLO mensaje oculto (sin parte pública):

1. Activá "Agregar mensaje oculto cifrado"
2. Dejá VACÍO el campo "Mensaje público"
3. Escribí solo en "Mensaje oculto" + password
4. Presioná [ postear ]

→ No aparece nada en el timeline público
→ Solo existe para quien conozca la semilla
→ Completamente invisible


LEER MENSAJES OCULTOS
======================

DESCIFRAR UN MENSAJE
--------------------
1. Escribí la contraseña en el campo "semilla..."
2. Presioná Enter o el botón →
3. Si es correcta: aparece el mensaje
4. Si es incorrecta: "Password incorrecto"

No hay forma de saber qué mensajes tienen contenido oculto.
Solo quien conozca la contraseña puede descubrirlo.

CONVERSACIONES SECRETAS GRUPALES
---------------------------------
Cuando varias personas usan la MISMA contraseña,
se crea un "canal secreto" invisible.

Ejemplo:
- Ana postea mensaje oculto con contraseña "gato verde luna"
- Carlos postea mensaje oculto con contraseña "gato verde luna"
- María postea mensaje oculto con contraseña "gato verde luna"

Cuando alguien ingresa "gato verde luna":
→ Se descifran TODOS los mensajes con esa contraseña
→ Aparecen juntos, en orden cronológico
→ Cada uno con su usuario de color y barra lateral

Podés mezclar:
- Post público visible + mensaje oculto
- Post 100% invisible (solo oculto)

DIFERENCIA IMPORTANTE
---------------------
- Palabras de IDENTIDAD: crean tu usuario permanente
- Contraseñas de MENSAJES: cifran contenido oculto

Son DOS cosas distintas. No las confundas.


HERRAMIENTAS
============

FORMATO DE TEXTO
----------------
*texto*              → negrita
_texto_              → cursiva
<nombre:url.com>     → enlace clickeable
#hashtag             → etiqueta destacada

Ejemplo:
Esto es *importante* y _urgente_.
Más info: <aquí:ejemplo.com> #documentos

BUSCAR MENSAJES
---------------
El buscador encuentra mensajes en toda la base de datos.

- Una palabra: "documentos"
- Varias palabras: "documentos, firmados" (deben estar AMBAS)
- Números: "23" (busca en fecha, hora, usuario, contenido)
- No distingue mayúsculas/minúsculas

ACTUALIZAR
----------
Presioná [ actualizar ] para ver mensajes nuevos sin recargar.

TEMAS Y MODO NOCHE
------------------
[ tema ] → Cambia el estilo visual
[ día/noche ] → Alterna modo claro/oscuro

SALIR
-----
Presioná [ salir ] para cerrar tu sesión.
Tu identidad se borra del navegador.

Para volver: ingresá tus palabras de identidad nuevamente.


USOS PRÁCTICOS
==============

IDENTIDAD PERSISTENTE
---------------------
Usuario "meno.34.rite.87" comenta sobre plantas.
Otra persona pregunta algo relacionado.
Días después, "meno.34.rite.87" responde.
→ Se reconoce la continuidad sin revelar identidad

FIRMAR MENSAJES
---------------
Público: "Vendo bici rodado 26, buen estado"
Oculto: "Contacto: Juan - Tel: 1234-5678"
Contraseña: "bici26venta"
→ Solo quien conozca la contraseña ve tus datos

COORDINAR REUNIONES
-------------------
Público: "Reunión grupal miércoles 18hs"
Oculto: "Lugar: Café Central, Mitre y Sarmiento"
Contraseña: "reunion2026"
→ Anuncio público, detalles privados

CANAL SECRETO DE ESTUDIO
-------------------------
Ana: Oculto "Resumen capítulo 3: fotosíntesis es..."
Carlos: Oculto "Agregando: fórmula química C6H12O6"
María: Oculto "Ejercicio 5 resuelto: A) verdadero..."
Contraseña compartida: "biologia examen"
→ Conversación de estudio invisible en timeline público

MÚLTIPLES IDENTIDADES
----------------------
Palabras "gato mesa luz" → usuario "nana.12.yija.87"
Usás este usuario para temas de trabajo.

Palabras "sol rio viento" → usuario "reto.44.misi.91"
Usás este usuario para temas personales.

→ Dos identidades separadas, ambas anónimas


SEGURIDAD Y PRIVACIDAD
======================

CIFRADO AES-256
---------------
Mismo estándar que bancos y apps de mensajería segura.

Con 3-4 palabras: romper el cifrado requiere años
de procesamiento con supercomputadoras.

CÓMO FUNCIONA
-------------
1. Escribís mensaje + contraseña en TU navegador
2. TU navegador cifra el mensaje (AES-256)
3. Se envía al servidor SOLO la versión cifrada
4. El servidor guarda texto ilegible

El servidor NUNCA ve:
- Tus palabras de identidad
- Tus contraseñas de mensajes
- El contenido de mensajes ocultos

QUÉ NO GUARDAMOS
----------------
- Palabras de identidad
- Contraseñas de mensajes
- Direcciones IP
- Datos del navegador
- Cookies de rastreo
- Ubicación geográfica
- Historial de búsquedas

QUÉ SÍ GUARDAMOS
----------------
- Mensajes públicos (texto plano)
- Mensajes ocultos (cifrados, ilegibles)
- Usuario generado (ej: meno.34.rite.87)
- Fecha y hora de publicación
- Color del usuario (generado automáticamente)

TODO SE BORRA CADA LUNES 00:00 HS
----------------------------------
Máximo de vida: 7 días.

El sitio muestra un contador con tiempo restante
hasta la próxima purga.


LÍMITES TÉCNICOS
================

CARACTERES
----------
- Mensaje público: 576 caracteres máximo
- Mensaje oculto: 288 caracteres máximo
- Palabras de identidad: 3-12 palabras
- Contraseñas de mensajes: ilimitadas (óptimo 1-12 palabras)

IDENTIDAD
---------
- Mínimo 3 palabras (menos es muy inseguro)
- Máximo 12 palabras (más es difícil recordar)
- Recomendado: 3-5 palabras

SIN EDICIÓN
-----------
No se pueden editar ni borrar mensajes.
Solo desaparecen el lunes 00:00hs.

SIN MULTIMEDIA
--------------
Solo texto. No se aceptan:
- Imágenes
- Videos
- Archivos adjuntos


FILOSOFÍA DEL PROYECTO
======================

¿POR QUÉ EXISTE POSTE.AR?
--------------------------
Porque no todo necesita ser permanente.
Porque el anonimato puede ser liberador.
Porque queremos comunicarnos sin dejar rastro.

Experimento sobre identidad efímera autogestionada:
- Tu identidad la controlás vos (tus palabras)
- No hay empresa que guarde tus datos
- No hay recuperación de contraseña (no la guardamos)
- Anonimato real, no marketing

En un mundo donde todo se registra y vende,
poste.ar es un espacio que se borra solo.

No hay algoritmos. No hay perfiles. No hay permanencia.
Solo comunicación, aquí y ahora.

STACK TÉCNICO
