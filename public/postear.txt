POSTE.AR - MANUAL DE USO
========================

Red social anónima y efímera. Sin usuarios, sin IPs, sin rastros.
Todo se borra cada lunes a las 00:00 hs.


CÓMO PUBLICAR
=============

MENSAJE PÚBLICO
---------------
1. Escribí tu mensaje (máximo 576 caracteres)
2. Presioná [ postear ]
3. Aparece en el timeline con tu etiqueta temporal

MENSAJE OCULTO CIFRADO
-----------------------
Para agregar un mensaje secreto a tu publicación:

1. Activá "Agregar mensaje oculto cifrado"
2. Escribí tu mensaje secreto (máximo 288 caracteres)
3. Escribí una contraseña en "Password"
   Ejemplo: "perro mesa luz" (3-5 palabras es óptimo)
4. Presioná [ postear ]

El mensaje público se ve normalmente.
El mensaje oculto queda cifrado (ilegible para todos).

MENSAJES INVISIBLES
--------------------
Podés postear SOLO mensaje oculto (sin parte pública):

1. Activá "Agregar mensaje oculto cifrado"
2. Dejá VACÍO el campo "Mensaje público"
3. Escribí solo en "Mensaje oculto" + password
4. Presioná [ postear ]

→ No aparece nada en el timeline público
→ Solo existe para quien conozca la semilla
→ Completamente invisible para el resto


LEER MENSAJES OCULTOS
======================

DESCIFRAR UN MENSAJE
--------------------
Todos los mensajes tienen un campo "semilla..." al pie.

Si conocés la contraseña de un mensaje oculto:
1. Escribí la contraseña en el campo "semilla..."
2. Presioná Enter o el botón →
3. Si es correcta: aparece el mensaje (fondo gris claro)
4. Si es incorrecta: "Password incorrecto"

IMPORTANTE: No hay forma de saber qué mensajes tienen
contenido oculto. Todos se ven iguales.
Solo quien conozca la semilla puede descubrirlo.

CONVERSACIONES SECRETAS GRUPALES
---------------------------------
Cuando varias personas usan la MISMA semilla, se crea
un "canal secreto" invisible.

Ejemplo:
- Ana postea mensaje oculto con semilla "gato verde luna"
- Carlos postea mensaje oculto con semilla "gato verde luna"
- María postea mensaje oculto con semilla "gato verde luna"

Cuando alguien ingresa "gato verde luna" en cualquier post:
→ Se descifran TODOS los mensajes con esa semilla
→ Aparecen juntos, en orden cronológico
→ Cada uno con su etiqueta de color y barra lateral

Esto permite:
- Conversaciones grupales cifradas
- Canales secretos por tema
- Hilos invisibles dentro del timeline público

Podés mezclar mensajes públicos + ocultos:
- Post público visible para todos
- Post oculto solo para el grupo secreto
- Post 100% invisible (solo oculto)

CONTRASEÑAS (SEMILLAS)
----------------------
Podés usar entre 1 y 12 palabras.

Ejemplos:
- "perro" (poco seguro)
- "perro tiza golf" (seguro)
- "perro tiza golf maria luna casa..." (muy seguro)

Recomendación: 3-5 palabras fáciles de recordar.
Más palabras = más seguridad.


TU IDENTIDAD TEMPORAL
=====================

ETIQUETA
--------
Tu identificador anónimo temporal durante esta sesión.
Formato: nana.12.yija.87 (sílabas + números)

Mientras no cierres la página o presiones [ reetiquetar ],
todos tus mensajes llevan la misma etiqueta.

Permite:
- Mantener conversaciones sin revelar identidad
- Que otros reconozcan tus aportes
- Abandonar la identidad cuando quieras (un click)

REETIQUETAR
-----------
Presioná [ reetiquetar ] para cambiar tu etiqueta.
Tu nueva etiqueta es única e irrepetible.

AVATAR
------
El cuadrado de colores con dos letras es tu avatar automático.
Se genera a partir de tu etiqueta.

Ayuda a reconocer visualmente tus mensajes sin leer
la etiqueta completa cada vez.


HERRAMIENTAS
============

FORMATO DE TEXTO
----------------
*texto*              → negrita
_texto_              → cursiva
<nombre:url.com>     → enlace clickeable
#hashtag             → etiqueta destacada

Ejemplo:
Esto es *importante* y _urgente_.
Más info: <aquí:ejemplo.com> #documentos

BUSCAR MENSAJES
---------------
El buscador encuentra mensajes en toda la base de datos,
no solo los 50 más recientes del timeline.

- Una palabra: "documentos"
- Varias palabras: "documentos, firmados" (deben estar AMBAS)
- Números: "23" (busca en fecha, hora, etiqueta, contenido)
- No distingue mayúsculas/minúsculas

ACTUALIZAR
----------
Presioná [ actualizar ] para ver mensajes nuevos sin recargar (F5).

TEMAS Y MODO NOCHE
------------------
[ tema ] → Cambia el estilo visual (default, retro, mínimo, ascii)
[ día/noche ] → Alterna entre modo claro y oscuro


USOS PRÁCTICOS
==============

FIRMAR MENSAJES
---------------
Público: "Vendo bici rodado 26, buen estado"
Oculto: "Contacto: Juan - Tel: 1234-5678"
Semilla: "bici26venta"
→ Solo quien conozca la semilla ve tus datos

COORDINAR REUNIONES
-------------------
Público: "Reunión grupal miércoles 18hs"
Oculto: "Lugar: Café Central, Mitre y Sarmiento"
Semilla: "reunion2026"
→ Anuncio público, detalles privados

CANAL SECRETO DE ESTUDIO
-------------------------
Ana: Oculto "Resumen capítulo 3: fotosíntesis es..."
Carlos: Oculto "Agregando: fórmula química C6H12O6"
María: Oculto "Ejercicio 5 resuelto: A) verdadero..."
Semilla compartida: "biologia examen"
→ Timeline público normal, conversación de estudio invisible

INFORMACIÓN SENSIBLE
--------------------
Público: "Disponible info sobre trámites"
Oculto: "DNI, partida nacimiento, comprobante domicilio..."
Semilla: (acordada previamente)
→ Lista completa solo para interesados

JUEGO DE ADIVINANZAS
---------------------
Público: "Animal de cuatro patas, ladra y es fiel"
Oculto: "¡Correcto! La respuesta era: perro"
Semilla: "perro"
→ Solo quien adivine puede descifrar la confirmación

COMUNICACIÓN PRIVADA EN PÚBLICO
--------------------------------
Ana: Público "Sobre lo que hablamos..."
     Oculto "Confirmo, traigo los documentos mañana"
Carlos: Público "Entendido"
        Oculto "Perfecto, nos vemos 15hs en el lugar"
Semilla compartida: "proyecto2026"
→ Para observadores: conversación genérica
→ Para ustedes: comunicación cifrada completa


SEGURIDAD Y PRIVACIDAD
======================

CIFRADO AES-256
---------------
Mismo estándar que:
- Bancos para transacciones online
- Apps de mensajería segura (Signal, WhatsApp)
- Gobiernos para datos clasificados

Con 3-4 palabras: romper el cifrado requiere años
de procesamiento con supercomputadoras.

Con 12 palabras: prácticamente imposible (siglos).

CÓMO FUNCIONA
-------------
1. Escribís mensaje + contraseña en TU navegador
2. TU navegador cifra el mensaje (AES-256)
3. Se envía al servidor SOLO la versión cifrada
4. El servidor guarda texto ilegible

El servidor NUNCA ve:
- Tu contraseña
- El contenido del mensaje oculto

Solo vos y quienes conozcan la contraseña pueden leerlo.

QUÉ NO GUARDAMOS
----------------
- Contraseñas de mensajes ocultos
- Direcciones IP
- Datos del navegador
- Cookies de rastreo
- Ubicación geográfica
- Historial de búsquedas
- Identidad de usuarios (no hay usuarios)

QUÉ SÍ GUARDAMOS
----------------
- Mensajes públicos (texto plano)
- Mensajes ocultos (cifrados, ilegibles)
- Etiqueta temporal del mensaje
- Fecha y hora de publicación
- Color del avatar (generado automáticamente)

TODO SE BORRA CADA LUNES 00:00 HS
----------------------------------
No importa si un mensaje tiene 5 minutos o 6 días:
al llegar el lunes, el timeline completo se vacía.

Máximo de vida: 7 días.

El sitio muestra un contador con tiempo restante
hasta la próxima purga.


LÍMITES TÉCNICOS
================

CARACTERES
----------
- Mensaje público: 576 caracteres máximo
- Mensaje oculto: 288 caracteres máximo
- Contraseña: ilimitada (pero 1-12 palabras es óptimo)

CAPACIDAD
---------
Base de datos: ~10000 mensajes aprox.
Si se llena antes del lunes, los mensajes más antiguos
se eliminan automáticamente.

SIN EDICIÓN
-----------
No se pueden editar ni borrar mensajes una vez publicados.
Solo desaparecen:
- El lunes 00:00hs (purga semanal)
- Si la base se llena (se borran los más viejos)

SIN MULTIMEDIA
--------------
Solo texto. No se aceptan:
- Imágenes
- Videos
- Archivos adjuntos


FILOSOFÍA DEL PROYECTO
======================

¿POR QUÉ EXISTE POSTE.AR?
--------------------------
Porque no todo necesita ser permanente.
Porque el anonimato puede ser liberador.
Porque a veces queremos comunicarnos sin dejar rastro.

Experimento sobre comunicación efímera, inspirado en
los tablones de anuncios del barrio: alguien pega un aviso,
alguien lo lee, la lluvia lo borra.

En un mundo donde todo se registra, archiva y vende,
poste.ar es un espacio que se borra solo.

No hay algoritmos. No hay perfiles. No hay permanencia.
Solo comunicación, aquí y ahora.


TECNOLOGÍA
==========

CÓDIGO ABIERTO
--------------
El código completo es público y auditable.
Cualquiera puede verificar cómo funciona, revisar
la seguridad o crear su propia versión.

Presioná [ source ] para acceder al repositorio.

STACK TÉCNICO
-------------
- Frontend: HTML, CSS, JavaScript vanilla
- Backend: Node.js + Express
- Base de datos: LibSQL (SQLite remoto)
- Cifrado: CryptoJS (AES-256)
- Hosting: Cloud

Minimalista. Sin dependencias innecesarias.

INFRAESTRUCTURA
---------------
Dos servidores independientes:

1. Servidor web: interfaz, búsquedas, purga semanal
2. Servidor DB: almacena mensajes públicos y cifrados

Separación para mejor seguridad y rendimiento.


ADVERTENCIAS
============

- Usá poste.ar bajo tu propia responsabilidad
- No publiques contenido ilegal
- No publiques información que viole privacidad de terceros
- El anonimato no te protege de consecuencias legales
- Los administradores NO son responsables del contenido
- Este servicio puede discontinuarse sin previo aviso

Si olvidás una contraseña, el mensaje se pierde para SIEMPRE.
No hay recuperación posible.


SOPORTE
=======

Proyecto experimental sin fines de lucro.
Sin equipo de soporte ni servicio al cliente.
Funciona "tal cual es" (as-is).

Para reportar problemas técnicos graves o vulnerabilidades,
utilizá el buscador para encontrar información de contacto
publicada por la comunidad.
