POSTE.AR - CÓDIGO FUENTE
=========================

Proyecto open source de comunicación anónima y efímera.

REPOSITORIO
-----------
[Próximamente disponible en GitHub]

TECNOLOGÍAS
-----------
- Backend: Node.js + Express
- Base de datos: Turso (libSQL)
- Frontend: HTML + CSS + Vanilla JavaScript

CARACTERÍSTICAS
---------------
- Publicación anónima con marcas temporales
- Búsqueda en toda la base de datos
- Purga automática cada 7 días
- Sin registro de usuarios
- Sin almacenamiento de IPs

FILOSOFÍA
---------
Comunicación efímera, anónima y sin rastro permanente.
Un tablón digital para mensajes que no necesitan eternidad.
